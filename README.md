# Smart Task Manager

A simple React Task Management application built for practice and portfolio.

## Tech Stack
- React.js
- JavaScript
- CSS
- Vite

## Features
- Add Tasks
- Delete Tasks
- Mark as Completed
- Filter Tasks (All / Completed / Pending)
- LocalStorage Save (Data stays after refresh)

##  Purpose
This project was created to improve React skills:
- useState
- useEffect
- Event Handling
- Filtering Logic
